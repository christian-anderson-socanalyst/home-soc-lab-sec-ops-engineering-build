# 🔐 Home SOC Lab Build
Four (4) years in security operations in an SEC regulated environment. Hands-on work in a live cyber range. A stack of certs with more on the way.

And yet, here I am building a home lab. 😄

Not because I need to learn what Wazuh is. Because there is real value in owning infrastructure from the ground up. Designing the network, making the architecture decisions, breaking things at 11pm and figuring out why. That kind of ownership is different from working inside someone else's environment, and it shows in ways that are hard to fake.

My repo documents the full build: network segmentation, hypervisor setup, SIEM deployment, and attack/defend exercises against intentionally vulnerable targets. The wins, the wrong turns, and the reasoning behind every decision along the way.

Built to showcase. Documented to share. 🔧
